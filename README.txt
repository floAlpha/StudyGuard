#速记功能
1.速记功能支持保存文字笔记和截图笔记
2.任何时候按下F1键可以打开速记录入界面即可录入文字
3.如果要保存截图，只需截图后点击屏幕上蓝色字体即可保存
4.屏幕上会显示速记回顾悬浮窗，按下CTRL+SHIFT+Z开关悬浮窗
5.悬浮窗会随机显示已保存的笔记和截图，点击图片打开原图，双击文字和图片不再显示该条记录

#备忘录功能
1.按下F2打开备忘录录入功能，选择时间和备忘任务然后保存
2.到达备忘录中任务的时间时，系统会准时发出Win10通知

#任务计划功能
1.支持按下F3录入每日任务项
2.到达时间工具会询问是否按时接受任务或者推迟取消任务
3.任务列表栏会显示任务详情和完成情况
